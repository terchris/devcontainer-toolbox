## General Icon Sources

These sites provide large collections of tech brand icons:

1. **[Simple Icons](https://simpleicons.org/)** - 3000+ tech brand SVGs with official colors
2. **[Devicon](https://devicon.dev/)** - Programming language and dev tool icons
3. **[Heroicons](https://heroicons.com/)** - Generic UI icons (MIT license)
4. **[CNCF Artwork](https://github.com/cncf/artwork)** - All CNCF project logos
5. **[Wikimedia Commons](https://commons.wikimedia.org/)** - Many tech logos with clear licensing

## Download Instructions

1. Visit the source URL for each logo
2. Download in SVG format when available (PNG as fallback)
3. Save to `website/static/img/tools/src/` for tool logos
4. Save to `website/static/img/categories/src/` for category logos
5. Use naming convention: `<logo-id>.<ext>` (e.g., `dev-python-logo.svg`)

The `dev-logos.sh` script will process all source images to 512x512 WebP format.
