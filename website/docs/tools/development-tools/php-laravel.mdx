## VS Code Extensions

| Extension | Description |
|-----------|-------------|
| [PHP Intelephense](https://marketplace.visualstudio.com/items?itemName=bmewburn.vscode-intelephense-client) | Advanced PHP language support with IntelliSense |
| [PHP Debug](https://marketplace.visualstudio.com/items?itemName=xdebug.php-debug) | Debug PHP applications using Xdebug |
| [PHP DocBlocker](https://marketplace.visualstudio.com/items?itemName=neilbrayfield.php-docblocker) | Automatically generate PHPDoc comments |
| [Composer](https://marketplace.visualstudio.com/items?itemName=ikappas.composer) | Composer dependency manager integration |
| [PHP Namespace Resolver](https://marketplace.visualstudio.com/items?itemName=mehedidracula.php-namespace-resolver) | Auto-import and resolve PHP namespaces |
| [Laravel Blade Snippets](https://marketplace.visualstudio.com/items?itemName=onecentlin.laravel-blade) | Blade syntax highlighting and snippets |
| [Laravel Artisan](https://marketplace.visualstudio.com/items?itemName=ryannaddy.laravel-artisan) | Run Laravel Artisan commands from VS Code |

## Installation

Install via the interactive menu:

```bash
dev-setup
```

Or install directly:

```bash
.devcontainer/additions/install-dev-php-laravel.sh
```
