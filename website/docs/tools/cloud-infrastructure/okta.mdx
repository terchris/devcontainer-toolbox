## Python Packages (pip)

| Package | Description |
|---------|-------------|
| [okta-cli](https://pypi.org/project/okta-cli) | Okta command-line interface for identity management |

## VS Code Extensions

| Extension | Description |
|-----------|-------------|
| [Okta Explorer](https://marketplace.visualstudio.com/items?itemName=OktaDcp.okta-explorer) | Browse and manage Okta organizations, users, and groups |

## Installation

Install via the interactive menu:

```bash
dev-setup
```

Or install directly:

```bash
.devcontainer/additions/install-tool-okta.sh
```
