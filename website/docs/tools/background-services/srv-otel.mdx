OpenTelemetry Collector Contrib with script_exporter for comprehensive devcontainer monitoring. Collects custom metrics via shell scripts, supports Prometheus/Loki/Tempo exporters, and provides resource processors for multi-tenant tagging.
:::

## Installation

Install via the interactive menu:

```bash
dev-setup
```

Or install directly:

```bash
.devcontainer/additions/install-srv-otel-monitoring.sh
```
