Language-agnostic development utilities including SQLTools for database management (MySQL, PostgreSQL, SQLite, MSSQL, MongoDB), REST Client for HTTP API testing, and Docker extension for container, image, and volume management.
:::

## System Packages

| Package | Description |
|---------|-------------|
| [docker.io](https://packages.debian.org/bookworm/docker.io) | Docker CLI for container management |

## VS Code Extensions

| Extension | Description |
|-----------|-------------|
| [SQLTools](https://marketplace.visualstudio.com/items?itemName=mtxr.sqltools) | Database management and SQL query tool for MySQL, PostgreSQL, SQLite, MSSQL, MongoDB |
| [REST Client](https://marketplace.visualstudio.com/items?itemName=humao.rest-client) | Send HTTP requests and view responses directly in VS Code |
| [Docker](https://marketplace.visualstudio.com/items?itemName=ms-azuretools.vscode-docker) | Manage containers, images, volumes, networks, and Dockerfiles |

## Installation

Install via the interactive menu:

```bash
dev-setup
```

Or install directly:

```bash
.devcontainer/additions/install-tool-dev-utils.sh
```
