- **Isometric Cube**: Container/dev environment
- **Code Brackets `< >`**: Development and coding

## Generating Assets from logo.svg

Once the SVG is finalized, use these commands inside the devcontainer to generate other required assets.

### Prerequisites

The devcontainer has ImageMagick installed via `install-dev-imagetools.sh`. Run these commands inside the devcontainer.

### 1. Generate Favicon (multi-size ICO)

```bash
convert /workspace/website/static/img/logo.svg \
  -resize 256x256 \
  -define icon:auto-resize=256,128,64,48,32,16 \
  /workspace/website/static/img/favicon.ico
```

### 2. Generate PNG for Social Media / Fallback

```bash
# Standard size (512x512)
convert /workspace/website/static/img/logo.svg \
  -resize 512x512 \
  /workspace/website/static/img/logo.png

# Large size for social cards (1200x630 with padding)
convert /workspace/website/static/img/logo.svg \
  -resize 400x400 \
  -gravity center \
  -background white \
  -extent 1200x630 \
  /workspace/website/static/img/social-card.png
```

### 3. Generate Apple Touch Icon

```bash
convert /workspace/website/static/img/logo.svg \
  -resize 180x180 \
  /workspace/website/static/img/apple-touch-icon.png
```

### 4. Generate Android Chrome Icons

```bash
# 192x192
convert /workspace/website/static/img/logo.svg \
  -resize 192x192 \
  /workspace/website/static/img/android-chrome-192x192.png

# 512x512
convert /workspace/website/static/img/logo.svg \
  -resize 512x512 \
  /workspace/website/static/img/android-chrome-512x512.png
```

## Typography

The website uses the default Docusaurus fonts:
- **Headings**: System font stack (San Francisco, Segoe UI, etc.)
- **Body**: System font stack
- **Code**: Monospace (Consolas, Monaco, etc.)

---

## Usage Guidelines

### Do
- Use the SVG logo when possible for crisp rendering
- Maintain adequate spacing around the logo
- Use on backgrounds with sufficient contrast

### Don't
- Stretch or distort the logo
- Change the logo colors
- Add effects (shadows, gradients) to the logo
- Use low-resolution versions when SVG is available

---

## Mission Alignment

DevContainer Toolbox is part of the [SovereignSky](https://sovereignsky.no) initiative, promoting digital sovereignty for Norway. The branding reflects:

- **Security**: Shield symbolizes protection of development environments
- **Containerization**: Cube represents isolated, portable dev environments
- **Open Development**: Code brackets represent the developer-first approach
